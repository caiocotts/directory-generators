To use these, put them in the directory you'd like the structure to be generated in and run the script.

The script will then create the following structure:

```
.
└── comp1234
    ├── assignments
    │   ├── assignment01
    │   │   └── assets
    │   │       ├── css
    │   │       └── img
    │   ├── assignment02
    │   │   └── assets
    │   │       ├── css
    │   │       └── img
    │   └── assignment03
    │       └── assets
    │           ├── css
    │           └── img
    └── lab
        ├── exercises
        │   ├── wk01
        │   ├── wk02
        │   ├── wk03
        │   ├── wk04
        │   ├── wk05
        │   ├── wk06
        │   ├── wk07
        │   ├── wk08
        │   ├── wk09
        │   ├── wk10
        │   ├── wk11
        │   ├── wk12
        │   ├── wk13
        │   └── wk14
        └── practices
            ├── wk01
            ├── wk02
            ├── wk03
            ├── wk04
            ├── wk05
            ├── wk06
            ├── wk07
            ├── wk08
            ├── wk09
            ├── wk10
            ├── wk11
            ├── wk12
            ├── wk13
            └── wk14
```

To be clear, the `.sh` is for UNIX-like systems, and the `.bat` is for windows exclusively.
